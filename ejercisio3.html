Ejercisio 3: Etiquetas de contenido y semántica

En esta pagina el estudiante practicara etiquetas semánticas de texto y también algunas etiquetas personalizadas en el curso.

Texto insertado y eliminado

Este parrafo contiene texto recientemente agregado texto que fue eliminado para ilustrar cambios editoriales. Además, se incluyen etiquetas personalizadas. INCS (custom) y DELS (custom).

Abreviaturas, acrónimos y definiciones

El termino HTML es un lenguaje de marcado empleado para estructurar documentos que los navegadores pueden interpretar. En contextos informales, se usan acrónimos como ASAP para expresar "lo antes posible"

Lista de definiciones

En HTML existe una estructura especial para listar terminos y definiciones: <d1>

HTML
Lenguaje de marcado utilizado para estructurar el contenido en la web.
CSS
Lenguaje de estilos en cascada que define la presentacion visual de los documentos HTML.
JavaScript
Lenguaje de programacion que permite agregar interactividad y logica a las paginas web.

Cita en bloque: (blockquote)

"El elemento <blockquote> se utiliza para denotar una cita extensa. Suele renderizarse con sangria y separación del texto principal."

Observa como el navegador presenta la cita con un formato diferenciado, lo cual mejora la calidad del contenido citado frente al contenido propio.
